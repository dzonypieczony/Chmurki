# Pogoda na każdy dzień

Nasza aplikacja została stworzona z myślą o zapewnieniu przyjemnego i czytelnego sprawdzania informacji pogodowych w danych lokacjach

## Funkcjonalności
Nasza aplikacja pozwala na:
- Wyszukiwanie dokładnej lokacji
- Przeglądanie danych pogodowych na aktualną godzinę
- Przeglądanie maksymalnych wartości parametrów pogodowych na aktualny dzień
- Przeglądanie prognozy pogody na każdy dzień 14 dni do przodu
- Zgłaszanie błędów, propozycji lub własnych wiadomości za pomocą formularza kontaktowego

## Technologie:
Aplikacja została utworzona za pomocą:
- JavaScript
- CSS
- HTML

## Dane i grafika
API: https://open-meteo.com/en/docs <br>
